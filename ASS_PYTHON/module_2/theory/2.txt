1. Common list operations: concatenation, repetition, membership. 

    concatenation : joint two or more list 
            ex : list1 = [1, 2, 3]
                 list2 = [4, 5, 6]
                 res = list1 + list2
                 print(res)

    repetition : by using "*" symbole we excute repetition of list
            ex : list1 = [1, 2, 3]
                 res = list1 * 3
                 print(res)

    membership : we can excute by "in" operations
            ex : list1 = [1, 2, 3, 4, 5]
                 print(3 in list1)
                 print(6 in list1)
                 print(6 not in list1)

2. Understanding list methods like append(), insert(), remove(), pop().

    append()        add list item
    insert()        add the element at specific position of list
    pop()           remove list item from last
    remove()        remove specific element form list
